io Chat

Real time websocket chat app using node, express, socket.io, vanilla JS DOM manipulation, and custom CSS. ioChat persists messages and has a Skype inspired design. It was built to practice Socket.io, a real-time event based bidirection communication library for Javascript. Chat messages left by app users are cached in a MongoDB server to allow a persisted history upon susequent sign in.
